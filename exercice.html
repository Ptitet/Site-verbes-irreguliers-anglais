Révises tes verbes irréguliers !

Ce site te permet de réviser facilement tous les verbes irréguliers anglais. Tu y trouveras une liste de tous les verbes, et un espace pour t'entrainer.

Sélectionnez un mode :

Les verbes doivent être écrits selon le format infinitif simple-past participe-passé comme suit : do did done

Cas spécial avec être: be was were been

Aide sur les différents modes d'entrainement :

  • Normal : Tous les verbes seront donnés dans un ordre aléatoire
  • Infini : Entraine toi jusqu'à la nuit des temps. Sauf si tu décides de t'arrêter
  • Temps imparti : Donne un maximum de réponses justes dans le temps imparti
  • Maximum d'essais : Choisi un nombre de tentatives et réalise-les le plus rapidement possible
